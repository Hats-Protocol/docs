# Hats Protocol Developer Documentation

This repository contains comprehensive documentation for developers working with the Hats Protocol ecosystem. The documentation is organized in two formats:

1. **Markdown Files**: Individual `.md` files organized in a hierarchical structure, providing detailed documentation for different components of the Hats Protocol.

2. **LLMs.txt**: A consolidated text file (`llms.txt`) that contains all the documentation in a format optimized for Large Language Models (LLMs) to consume. This file follows the structure used by other major platforms like Anthropic and Coinbase.

## Repository Structure

- `llms.txt`: Single-file documentation optimized for LLM consumption
- Markdown files organized by component:
  - Protocol Specifications
  - SDKs (v1, Modules, Account)
  - Subgraphs
  - Security Audits
  - Module Development Guides

## Purpose of llms.txt

The `llms.txt` file serves as a consolidated reference for AI agents and LLMs to better understand and assist with Hats Protocol development. It follows established conventions for LLM documentation, including:

- Clear table of contents
- Hierarchical organization of content
- Consistent formatting
- Cross-references between related sections

## Maintenance

When updating documentation:

1. Update the relevant markdown files
2. Regenerate `llms.txt` to maintain consistency
3. Ensure all cross-references are maintained
